V2.10.07	时间：2015.08.28	硬件：FC-ME1-V1 V1.0.2		修改：周雁

烧录文件：IGL1302-SA_V2.10.07_burn.hex
升级文件：IGL1302-SA_V2.10.07_update.dat
说    明：
		1. 第一次触摸默认点亮6个背景灯，不往外发送联动消息，屏幕显示上一次休眠前屏幕内容。

		5. 首次上电后直接显示上次状态；
		6. 增加电量低于8%的电量时，提示用户充电，同时3s后休眠，而不执行其他操作；
